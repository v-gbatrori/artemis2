HOW THIS WORKS

This display shows the real-time position of NASA's Orion spacecraft during the Artemis II mission.

Data sources (priority order):

  • NASA AROW LIVE — live telemetry from NASA's Artemis Real-time Orbit Website bucket. Updates every ~55s. Provides exact position, velocity, attitude quaternions, solar array angles and thruster data.
  • NASA OEM FILE — official ephemeris file published at nasa.gov. Contains thousands of position points at 4-minute intervals. Loaded once, used as high-accuracy trajectory.
  • PHYSICS MODEL — fallback when no live data is available. A mathematical approximation of the mission trajectory built from the mission profile.

Map legend:

  • Bright trail = Orion's path so far (gradient: blue → white)
  • Dim line = full planned trajectory
  • Dashed gold circle = Moon's orbital path
  • Gold trail = Moon's recent path (5-day history)
  • Rotating arc on Earth = radar sweep (visual only)

Time sync: When AROW live data is active, the MET clock is driven by the spacecraft's onboard timer (Parameter 5001), not your local clock. The Moon and Orion positions are computed for this mission-time, keeping everything in sync.

Smooth animation: Orion's icon is lerp-smoothed (factor 0.025/frame) toward the true computed position, eliminating jumps when new data arrives. When switching to OEM, positions blend gradually.

YouTube feed: Starts automatically muted (browser autoplay policy). Use the player controls to unmute. Click "Open on YouTube" to watch full-screen.
